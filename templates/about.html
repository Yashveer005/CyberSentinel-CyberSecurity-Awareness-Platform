🛡 About CyberSentinel

CyberSentinel is an interactive cybersecurity awareness platform designed to help students, beginners and professionals learn cybersecurity through practical tools, quizzes and real-world security concepts.

🎯 Our Mission

Our mission is to make cybersecurity simple, practical and accessible for everyone. Instead of only reading theory, users can practice with security tools, solve quizzes, strengthen passwords and improve cyber awareness.

🚀 Platform Features

🧠 Interactive Quiz

Test cybersecurity knowledge through randomized quizzes.

🛠 Security Tools

Password Analyzer, URL Checker, SHA-256 Generator, DNS Lookup, File Hash Checker and more.

📚 Learning Hub

Learn cybersecurity from beginner to advanced with structured articles and practical examples.

🏆 Gamified Dashboard

Earn XP, level up, unlock achievements and track your cybersecurity progress.
