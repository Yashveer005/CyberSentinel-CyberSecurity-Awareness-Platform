# 🛡 CyberSentinel

CyberSentinel is a Cyber Security Awareness and Learning Platform developed using Python Flask.

## Features

- Login & Register
- Cyber Security Quiz
- Certificate Generation
- Password Strength Analyzer
- URL Safety Checker
- Responsive UI

## Technologies

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
